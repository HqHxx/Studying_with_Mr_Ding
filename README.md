## 快速开始

### 1. 配置 API（首次使用必需）

在「批量生成中心」顶部的 **模型 API 配置** 区域填写：

- **Base URL**：你的大模型 API 地址（如 `https://ark.cn-beijing.volces.com/api/v3`）
- **Fast Model**：轻量级模型（用于选词和翻译，如 `doubao-seed-2-0-mini-260215`）
- **Core Model**：强推理模型（用于文章重写，如 `deepseek-v3-2-251201`）
- **API Key**：你的 API 密钥

填写后点击 **保存/覆盖当前名称预设**，支持最多保存 10 组配置。

### 2. 生成学习文章

**模式一：随机题库抽取（推荐）**

1. 选择「随机题库抽取」
2. 设置生成篇数（1~N 篇）
3. 选择目标难度：CET-4 / CET-6 / 考研 / 托福
4. 选择文章题材：history / science
5. 点击 **开始生成**

软件会自动：
- 从对应词库中智能挑选 25~50 个适合语境的单词
- 从语料库选取未使用过的文章
- 调用大模型重写并植入目标词汇
- 生成中英双语对照的 Markdown + PDF
- 自动标记已学单词和已使用文章

**模式二：自定义文章**

1. 选择「自定义文章」
2. 粘贴你的英文原文（800~5000 字符，纯英文）
3. 选择目标难度词库
4. 点击 **开始生成**

软件会从你提供的文章中提取语境，植入对应难度的词汇。

### 3. 阅读历史归档

切换到「历史归档阅读」Tab：

- **刷新**：加载所有已生成的文章
- **预览**：在软件内查看文章内容
- **EPUB**：将所有文章打包为电子书（适合导入手机/平板阅读）⭐⭐⭐⭐⭐
- **排版**：使用最新排版引擎重新生成所有 PDF

## 学习进度管理

在「批量生成中心」下方的 **学习进度管理** 区域：

| 按钮 | 功能 |
|------|------|
| 重置单词进度 | 重置当前选中级别的已学单词为未学状态 |
| 重置文章记录 | 清空已使用文章记录，文章可再次被选中 |
| 重置全部 | 重置所有级别的单词 + 所有文章记录 |

> 所有重置操作均有二次确认弹窗，防止误触。

## 技术原理

软件采用 **三级流水线** 架构：

1. **智能选词**（Fast Model）：从候选词库中挑选最适合文章语境的 25~50 个单词
2. **深度重写**（Core Model）：将原文重写为 600~800 词的高质量英文长文，并无痕植入目标词汇
3. **逐段精译**（Fast Model）：将英文长文逐段翻译为中英双语对照格式

## 注意事项

- 生成文章需要消耗 API Token，请确保账户余额充足
- 首次启动会自动创建 `data/学习进度词库.db`，请勿手动删除
- PDF 采用护眼暗黑主题，适合夜间阅读
- 所有学习进度保存在本地，不会上传至任何服务器
